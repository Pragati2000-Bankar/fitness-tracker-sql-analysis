# Fitness Tracker SQL Analysis

## Project Overview
- This project builds a relational SQL database for tracking workout activity, calories burned, and body progress.

## Database Tables
- Users
- Workouts
- Workout_Log
- Progress_Stats

## Key Analysis
- Total calories burned by muscle group
- Body fat progress tracking
- Most frequent workouts
- Workout type distribution
- Most targeted muscle groups

## Tools Used
- MySQL
- Database Design

## Example Insight
- Identify which workouts burn the most calories and track user fitness progress over time.
